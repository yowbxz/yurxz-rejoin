# YURXZ REJOIN

Auto rejoin Roblox via ADB — Android APK

## Cara Build APK

1. Fork/upload repo ini ke GitHub
2. Pergi ke tab **Actions**
3. Klik **Build APK YURXZ REJOIN**
4. Klik **Run workflow** → **Run workflow**
5. Tunggu ~15-20 menit
6. Setelah selesai, klik run yang hijau → scroll bawah → **Download Artifacts** → **YURXZ-REJOIN-APK**

## File
- `main.py` — Entry point Kivy app
- `server.py` — Backend HTTP + ADB controller
- `buildozer.spec` — Config build APK
- `static/` — UI web (index.html dll)
